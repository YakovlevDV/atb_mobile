# Описание:

В демо проекте реализованы автотесты для Android приложения АТБ банк с использованием Java, TestNG, TestNG, Selenide

- КОГДА запускаем приложение на эмуляторе 
  ТОГДА отображаеся страница 'Политика безопасности'

- КОГДА на странице 'Политика безопасности' нажимаем 'Пропустить'
  ТОГДА отображаеся страница 'Вход по номеру телефона'

- ДАНО страница 'Вход по номеру телефона'
  КОГДА вводим корректный номер и принимаем условия универсального банковского обслуживания
  ТОГДА кнопка 'Войти' активна"

- КОГДА на странице 'Вход по номеру телефона' нажимаем 'условия универсального договора банковского обслуживания'
  ТОГДА отображаеся страница с документом 'Универсальный договор банковского обслуживания'

- КОГДА на странице 'Вход в приложение' нажимаем 'Курсы валют' 
  ТОГДА отображаеся страница 'Курсы валют'

# Требования:

Для запуска тестов требуется наличие ряда библиотек:
- Java 21
- Maven 3+
- Node 20+, npm 11+
- appium@2+
- appium driver uiautomator2@3+

Так как тесты для мобильного приложения, то для запуска потребуются
- Android Studio, Android Emulator (AVD)

# Запуск тестов:

Необходимо наличие предустановленного приложения на симуляторе. В приложении при запуске на симуляторе появляется страница предупреждение - "Политика безопасности", все тесты построены с учетом этого. Если запускать на реальном девайсе этой страницы не будет, что приведет к падению тестов.

````
mvn clean test
-DsuiteXmlFile=allTests.xml
